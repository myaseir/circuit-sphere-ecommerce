# Electronics Kits Store API

A FastAPI backend with Clean Architecture for an electronics kits e-commerce platform.

## Features
- Clean Architecture separation (Entities, Use Cases, Interfaces)
- MongoDB with Motor async driver
- RESTful API with proper HTTP status codes
- Input validation with Pydantic v2
- Structured logging
- CORS enabled
- Ready-to-use seed data
